# Restaurant Reviews API Server
---
## Architecture
---
Local server
- Node.js
Development platform
- Docker
## Getting Started
---
###### Install project dependancies
```
npm i
```
###### Install Docker
[Download Docker here](https://docs.docker.com/docker-for-mac/install/)
###### Run Docker
```
docker-compose up
```
###### Run nodemon
```
nodemon index.js
```
